# Roadmap

Current roadmap themes include:

- Richer Windows input support
- Additional theming and browser UX polish
- Accessibility validation and screen-reader test coverage
- More cross-language samples for the C ABI surface
- Continued packaging hardening for mobile and browser targets

## Design Direction

The repository already emphasizes:

- Stable interop contracts
- Predictable runtime packaging
- Clear diagnostics when a platform narrows behavior

Future work should preserve those properties rather than trading them for feature count alone.
